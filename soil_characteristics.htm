Soil Characteristics


Soil Characeteristics are displayed in this website using ESRI JavaScript Application Programme Interfaces (APIs). The map was created using ESRI's Arcmap and published on Argis Server Manager. The map was published in order to make it accessible for display in the website. ESRI provides proprietary technology with a wide range of advantages such as a wide range of products to choose from and numerous functionality for optimum results and provides value for money. In this example, we incorporated some JavaScript API codes to display different basemaps, located at the top right corner of the map to provide users with the opportunity to choose from different basemap options, as well as a legend to provide information about the soil characteristics in the Isle of Wight. This map may be useful in providing farmers in the Isle of Wight with information when considering suitable sites for performing agricultural activities.

Note: You may be required to allow certificate to be able to view the map.
