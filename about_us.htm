About us

Web Mapping can be utilized within a website to provide geographical information in the "about us" section of a company's website to provide the public with a visual representation of their location. In this example, we utilize some Google JavaScript Application Programming Interfaces (APIs) to provide locational information about the Isle of Wight council to the general public. In addition, a google kml file has been used to indicate the precise location of the building for easier identification. Google kml files can be used for different application purposes and in the 'Towns and Roads' page, it has been used with leaflet to provide information about some towns in the Isle of Wight. There are many Google Javascript APIs that add more functionality to the traditional locational maps and in this example we explore one to show pixel coordinates for the Isle of Wight Council.
