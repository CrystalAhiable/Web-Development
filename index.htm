Isle of Wight

This is a prototype website for the Isle of Wight council showcasing the potent tools encompassed in Geospatial Technologies to provide web mapping services. Web mapping services increase the funcionality of traditional stand-alone maps and allow for extensive levels of interaction with data. Incorporating them in websites is beneficial for providing geographic information.

  • Learn More

Image Mapping: With the image mapping technology, we are able to link sections of a photo to provide more information.

For instance, the towns below have been linked to a weather forecast website for weather information. Images could be linked to other websites as well as other webpages within a website. In this example, the towns have been linked to the BBC weather website. Click on a town on the map below to learn more:

Another intriguing technology is the integration of JavaScript Application Programming Interfaces (APIs) from other web map tools with html for location based services. The tabs below link to maps from different internet mapping solutions, both proprietary and open source, to provide some spatial information about the Isle of Wight. The decision to use a proprietary source or an open source is dependent on the needs of the client as well as the resources available. Although proprietary sources are considered to provide many advantages, with the right technical knowledge, open source web mapping tools provide a lot of benefits. For this project, a number of open source tools will be considered.
