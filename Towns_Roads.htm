Towns and Roads


The Towns map has been incorporated into the website using the Leaflet JavaScript API and Google kml. The google kml which shows the towns as points were prepared using google earth's add placemark functionality and published to github to make it accessible in the website. The names of the towns can be determined by clicking on the markers. Leaflet which is a open source web mapping tool offers a variety of opportunities to display maps with different functionalities. In this example we have used some codes to incorporate a pop up, a marker and a circle covering the Newport area. Also, the latitude and longitude for each point in the map can be determined by clicking on the map. As an open source tool, it is an economical option for web mapping services. However, unlike other proprietary tools like ESRI products, its functionality can be restrictive or will require some extra technical knowhow. For instance in this example, an additional script for a plugin was required to add the google kml file.


In this second map, we illustrate the outcome of a different approach to web mapping using QGIS and Leaflet web mapping tools for incoporating maps in websites. Map preparation was done using the Quantum GIS (QGIS) tool and published as a leaflet web map using the qgis2web plugin. This web map has some interesting added functionality which is provided by the qgis2web plugin. On the left top corner of the map, users will be able to search for places on the map as well as to identify their current location. When users hover their mouse over the layer stack icon on the top right corner of the map, the legend is revealed to allow for interpretations to be made. For instance, the sizes of the triangles depict the population size for the towns. Also, for more details about the features such as the roads or the towns, users can hover their mouse over the features and a popup will appear displaying some information. This approach to web mapping is quite impressive and clients can get some level of sophistication at almost no cost.
