In this section we are displaying the possibilities that an approach using open source tools like geoserver and openlayers present in web mapping services. Geoserver is an open source web map server which allows users to publish data freely without having to acquire any license, making it a cheaper option compared to its proprietary competitors. Due to its interoperability features, it is compatible with data from a variety of sources and can be accessed in different formats. In this example, we showcase how Geoserver can be used to add maps to websites and here, we use a rainfall map which was prepared using the Quantum GIS (QGIS) open source software. Although Geoserver provides a wide range in terms of accessibility, Geoserver can be somewhat restrictive in its functionality, hence for a more sophisticated approach, other web servers such as ESRI's ArcGIS Enterprise is preferred. Also, it required more technical skills for its operation, which might make it a rather expensive approach.
